# Self-improving memory

Записи формата:
- <дата ISO> — <короткая цель/проблема>. <одно предложение вывода/правила/шаблона>.

Примеры:
- 2026-04-26 — Правило: перед изменением конфигураций читать AGENTS.md и SURFACE.md. Всегда проверять, изменится ли публичное поведение.
- 2026-04-26 — Исправление opencode wrapper: не вставлять лишний `--` между loader и бинарником. Если нужен прозрачный CLI, передавать argv напрямую и не ломать `acp`/`--help`.
