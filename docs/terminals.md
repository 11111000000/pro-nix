Terminals в pro-nix

Кратко: минималистичный, безопасный набор helper-функций для vterm; полная интеграция остаётся опциональной.

Включено
- `pro/vterm-yank` — вставить из kill-ring в vterm.
- `pro/vterm-interrupt` — отправить SIGINT в встраиваемый терминал.
- Небольшие helpers: выход из vterm-copy-mode (`C-g`), прокрутка/вход в copy-mode (`C-p` в vterm).

Как это работает
- Модуль безопасно проверяет наличие `vterm` и определяет функции только при доступности.
- Клавиши не навязываются глобально; модули регистрируют предложенные сочетания через механизм регистрации ключей.

TTY ограничения
- Некоторые модификаторы (AltGr/Right-Alt) не передаются терминалами — указывайте альтернативные сочетания в `emacs-keys.org`.

Проверка
- Headless: `emacs --batch -l emacs/base/init.el -f ert-run-tests-batch`.
