# Plan: Yggdrasil (IPv6 mesh overlay)

Цель
-----

Построить децентрализованную IPv6 overlay сеть между машинами pro‑nix без центрального control plane. Удобно для случаев, когда часто включены несколько хостов и требуется низкая латентность.

Шаги
-----

1) Установка

Установите `yggdrasil` на каждой машине. На NixOS это можно сделать через `environment.systemPackages` или соответствующий модуль.

2) Генерация конфигов

Генерируйте конфиг и ключи (`yggdrasil -genconf`) и распределите public части между машинами. Yggdrasil может использовать предварительно сгенерированные peer списка.

3) Запуск

Запустите демон `yggdrasil` и проверьте интерфейс (обычно `ygg0`) и IPv6‑адрес.

4) Использование

Подключайтесь по IPv6 адресам: `ssh user@[fdxx:...:...]`.

Преимущества
-------------

- Автономный mesh без центра.
- Низкая задержка, подходящая для постоянных соединений.

Ограничения
------------

- Требует распространения конфигурации peer‑to‑peer.
- IPv6 only в пределах overlay — может быть неудобно для интеграции с IPv4‑only средами.
