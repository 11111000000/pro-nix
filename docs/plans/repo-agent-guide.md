Кратко
-----
Этот документ служит кратким руководством для автоматических агентов и операторов,
которые взаимодействуют с репозиторием pro-nix. Он содержит ссылку на политику
и поясняет, какие файлы ожидаются в дереве источника.

Назначение
---------
Файл нужен как вспомогательный ресурс: некоторые Nix-модули и скрипты ссылаются
на `docs/plans/repo-agent-guide.md` для отображения ссылок, руководств или подсказок
в процессе сборки/активации. Отсутствие файла приводит к ошибкам типа
"path '.../docs/plans/repo-agent-guide.md' does not exist" при вычислении
объектов flake/nixos.

Содержание
---------
Этот файл — порождающийся текст; он может быть расширен в будущем. Текущее
состояние — минимальный placeholder, достаточный для восстановления ссылочной
целостности сборки.

Контракт
-------
- Путь `docs/plans/repo-agent-guide.md` должен существовать в репозитории.
- Содержимое может быть изменено без изменения публичной поверхности (SURFACE).

Автор: pro-nix agent
