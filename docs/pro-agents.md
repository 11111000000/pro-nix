# Описание поддерживаемых агентских инструментов

Кратко: перечисление инструментов, которые репозиторий поддерживает или оборачивает.

- goose — легковесный инструмент для локальной синхронизации/обработки подсказок (опционально).
- aider — локальный ассистент, запускаемый через pipx (в репозитории могут быть wrapper-скрипты).
- opencode — runtime и CLI для агентских задач; репо поставляет обёртки и опции для доставки.
- llm-lab — воспроизводимый JupyterLab контейнер/обёртка для исследований и E2E тестов.

Эти упоминания используются в тестах Proof для проверки наличия поддержки инструментов.
