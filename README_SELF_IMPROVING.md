Инструкция для агентов: self-improving

- Перед нетривиальной работой: загрузить ~/self-improving/memory.md и только самые маленькие релевантные файлы проекта.
- После значимых действий: добавить одну короткую запись в ~/self-improving/memory.md с форматом: <дата> — <цель>. <вывод>.

Файл self-improving/memory.md в репозитории служит шаблоном; агенты должны предложить создать локальную копию в ~/self-improving/memory.md, если таковой нет.
